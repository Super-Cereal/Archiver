# Archiver
  
  Архивирует папку или файл и складывает архив по заданному адресу.
Идет наглядное представление структуры папки в формате :
        /somepath
          /somedir1
            12 dirs;
            10 files;
          /somedir2
            0 dirs;
            2 files;
          somefile1 - 12 Kb
          somefile2 - 20 Mb
